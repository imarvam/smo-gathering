# تحديث قاعدة البيانات (Supabase)

لتنفيذ التغييرات المطلوبة على قاعدة البيانات، يرجى اتباع الخطوات التالية في لوحة تحكم Supabase:

## 1. تحديث قائمة الموظفين
قم بتشغيل الكود التالي في **SQL Editor** لمسح القائمة الحالية وإضافة الموظفين المحددين فقط:

```sql
-- مسح الموظفين الحاليين (تنبيه: سيتم حذف جميع البيانات الموجودة في جدول الموظفين)
TRUNCATE TABLE employees CASCADE;

-- إضافة القائمة الجديدة
INSERT INTO employees (name)
VALUES 
    ('عائشة الكواري'),
    ('شريفة المنصوري'),
    ('ريم المحمد'),
    ('مريم القوز'),
    ('ندى اليامي'),
    ('ياسمين العجي'),
    ('لولوه آل ثاني'),
    ('رامي أبو زيتون'),
    ('وصال العنزي'),
    ('فاطمة الكبيسي'),
    ('عائشة الكعبي'),
    ('مريم الغانم'),
    ('المها المالكي'),
    ('موضي الكواري'),
    ('نورة السويدي');
```

## 2. إعداد جدول الأسئلة (16 سؤال)
إذا لم يكن لديك جدول للأسئلة بعد، يمكنك إنشاء الجدول التالي وإضافة 16 خانة للأسئلة:

```sql
-- إنشاء جدول الأسئلة إذا لم يكن موجوداً
CREATE TABLE IF NOT EXISTS game_questions (
    id SERIAL PRIMARY KEY,
    question TEXT NOT NULL,
    options TEXT[] NOT NULL,
    correct_index INTEGER NOT NULL
);

-- إضافة 16 سؤال (قوالب جاهزة للتعديل)
INSERT INTO game_questions (question, options, correct_index)
SELECT 
    'السؤال رقم ' || i, 
    ARRAY['الخيار الأول', 'الخيار الثاني', 'الخيار الثالث', 'الخيار الرابع'], 
    0
FROM generate_series(1, 16) AS i;
```

> [!TIP]
> يمكنك تعديل الأسئلة مباشرة من خلال واجهة **Table Editor** في Supabase بعد تشغيل هذا الكود.
